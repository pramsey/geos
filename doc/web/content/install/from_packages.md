## Homebrew

[Homebrew](https://brew.sh/) is a package management system primarily for MacOS. The Homebrew package name is [geos](https://formulae.brew.sh/formula-linux/geos).

```sh
brew install geos
```

## MacPorts

[MacPorts](https://www.macports.org/) is a package management system for MacOS modelled on the FreeBSD "ports" system. The MacPorts package name is [geos](https://ports.macports.org/port/geos/summary).

```sh
port install geos
```
